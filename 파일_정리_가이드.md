# 파일 정리 가이드

## main으로 시작하는 파일들

### 1. `main.py` ⭐ (선택사항 - dlib 버전)
**역할**: dlib을 사용한 rPPG 측정 프로그램의 메인 실행 파일

**특징**:
- dlib 라이브러리 사용 (더 정확할 수 있음)
- Windows에서 설치가 복잡함 (CMake, Visual Studio Build Tools 필요)
- `shape_predictor_68_face_landmarks.dat` 파일 필요

**남겨둘지 여부**:
- ✅ **남겨두기**: dlib 버전을 사용하고 싶은 경우
- ❌ **삭제 가능**: MediaPipe 버전만 사용하는 경우

**권장**: MediaPipe 버전(`main_mediapipe.py`)을 사용하는 것이 더 쉽고 안정적이므로, dlib을 사용하지 않는다면 삭제해도 됩니다.

---

### 2. `main_mediapipe.py` ⭐⭐⭐ (필수 - 권장)
**역할**: MediaPipe를 사용한 rPPG 측정 프로그램의 메인 실행 파일

**특징**:
- MediaPipe 라이브러리 사용 (설치가 쉬움)
- Windows에서 바로 사용 가능
- 별도의 모델 파일 불필요
- 심박수 및 호흡률 측정
- MQTT 전송 지원

**남겨둘지 여부**:
- ✅ **반드시 남겨두기**: 실제로 사용하는 메인 프로그램

**권장**: 이 파일을 주로 사용하세요!

---

### 3. `mqtt_example.py` (선택사항 - 테스트/예제)
**역할**: MQTT로 전송되는 데이터를 수신하는 예제 프로그램

**특징**:
- rPPG 프로그램에서 전송하는 데이터를 확인하는 용도
- 디버깅 및 테스트에 유용
- 실제 측정에는 필요 없음

**남겨둘지 여부**:
- ✅ **남겨두기**: MQTT 데이터 수신 테스트가 필요한 경우
- ❌ **삭제 가능**: MQTT를 사용하지 않거나 테스트가 불필요한 경우

**권장**: MQTT를 사용한다면 남겨두는 것이 좋습니다. 데이터 수신 확인에 유용합니다.

---

### 4. `test_webcam.py` (선택사항 - 테스트/유틸리티)
**역할**: 웹캠 연결 및 작동 상태를 테스트하는 유틸리티

**특징**:
- 웹캠이 제대로 작동하는지 확인
- 카메라 해상도 및 FPS 확인
- 문제 해결에 유용

**남겨둘지 여부**:
- ✅ **남겨두기**: 웹캠 문제 해결이나 초기 설정 시 유용
- ❌ **삭제 가능**: 웹캠이 정상 작동하고 더 이상 필요 없는 경우

**권장**: 초기 설정이나 문제 해결 시 유용하므로 남겨두는 것을 권장합니다.

---

## 기타 유틸리티 파일들

### `list_cameras.py` (선택사항 - 유틸리티)
**역할**: 사용 가능한 카메라 목록 확인

**남겨둘지 여부**:

## 최소 구성 (권장)

실제로 사용하는 최소 구성:

1. ✅ **`main_mediapipe.py`** - 메인 프로그램 (필수)
2. ✅ **`rppg_mediapipe.py`** - rPPG 측정 모듈 (필수)
3. ✅ **`mqtt_client.py`** - MQTT 클라이언트 (MQTT 사용 시 필수)
4. ✅ **`camera_utils.py`** - 카메라 유틸리티 (필수)
5. ✅ **`requirements.txt`** - 패키지 목록 (필수)

선택사항:
- `test_webcam.py` - 웹캠 테스트 (유용)
- `mqtt_example.py` - MQTT 테스트 (MQTT 사용 시 유용)
- `list_cameras.py` - 카메라 목록 확인 (유용)

삭제 가능:
- `main.py` - dlib 버전 (MediaPipe만 사용 시)
- `rppg.py` - dlib 버전 (MediaPipe만 사용 시)
- `requirements-dlib.txt` - dlib 패키지 목록 (dlib 사용 안 할 시)
- `install_dlib_windows.md` - dlib 설치 가이드 (dlib 사용 안 할 시)
